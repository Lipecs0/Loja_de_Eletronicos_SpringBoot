Projeto colaborativo implementando uma API Restful utilizando Spring Boot. Criado para trabalho em aula de Programação Orientada a Objetos. A API gerencia recursos de uma loja de eletrônicos, permitindo operações CRUD sobre produtos e compras


Para acessar a página de produtos e compras, deve-se utilizar o Local Host conforme o exemplo abaixo:


exemplo1: localhost:nº/produtos.html
exemplo2: localhost:nº/shop.hyml
